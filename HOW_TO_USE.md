# Как использовать систему отправки запросов AI

## Быстрый старт

1. **Запустите сервер:**
   ```bash
   npm start
   ```

2. **Откройте браузер:**
   - Перейдите на `http://localhost:3091`
   - Или просто откройте `index.html`

3. **Отправьте запрос:**
   - Введите ваш запрос в поле ввода
   - Нажмите "Отправить запрос"
   - Запрос будет сохранен в файлы

4. **Обработка запросов AI:**

   После отправки запроса через форму, он сохраняется в два файла:
   - `user-request.txt` - простой текстовый файл с запросом
   - `NEW_REQUEST.txt` - файл-триггер с полной информацией

   **Чтобы AI обработал запрос, скажите ему одно из:**
   - "Прочитай NEW_REQUEST.txt и выполни запрос"
   - "Обработай запрос из user-request.txt"
   - "Выполни последний запрос пользователя"
   - "Прочитай user-request.txt и внеси изменения в проект"

## Примеры запросов

Вы можете отправлять любые запросы на изменение проекта:

- "Добавь валидацию email в форму"
- "Измени цвет кнопки на красный"
- "Создай новый компонент для отображения статистики"
- "Добавь функцию сохранения данных в localStorage"
- "Измени стили заголовка"

## Как это работает

1. Пользователь вводит запрос в форму
2. Запрос отправляется на сервер (POST /api/request)
3. Сервер сохраняет запрос в файлы:
   - `user-request.txt` - для быстрого чтения
   - `NEW_REQUEST.txt` - как триггер для AI
   - `last-request.json` - в формате JSON
   - `ai-requests/request-*.json` - для истории
4. AI читает файл и выполняет запрос
5. Проект изменяется согласно запросу

## Автоматизация

Если вы хотите автоматизировать процесс, можно создать скрипт, который будет:
- Мониторить файл `NEW_REQUEST.txt`
- Автоматически вызывать AI для обработки
- Удалять файл после обработки

Но в текущей версии нужно вручную попросить AI прочитать файл.
