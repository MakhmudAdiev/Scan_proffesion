# AI Помощник для Проекта 🤖

Веб-приложение, которое позволяет отправлять запросы AI через удобный интерфейс. AI обрабатывает эти запросы и вносит изменения в проект.

## Возможности

- ✨ Красивый современный интерфейс
- 📝 Поле ввода для запросов
- 📊 История запросов (сохраняется в браузере)
- 🔄 Обработка запросов через API
- 💾 Автосохранение истории

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

1. Запустите сервер:
```bash
npm start
```

2. Откройте браузер и перейдите на:
   - `http://localhost:3000` (если сервер запущен)
   - Или просто откройте `index.html` в браузере

## Использование

1. Введите ваш запрос в текстовое поле
2. Нажмите "Отправить запрос" или используйте `Ctrl+Enter`
3. AI обработает запрос и внесет изменения в проект
4. Просматривайте историю запросов внизу страницы

## Примеры запросов

- "Добавь кнопку для очистки формы"
- "Измени цвет фона на синий"
- "Создай новый компонент для отображения данных"
- "Добавь валидацию для поля ввода"

## API Endpoints

- `POST /api/request` - Отправка запроса
- `GET /api/status` - Статус сервера
- `GET /api/history` - История запросов

## Как работает интеграция с AI

1. Когда вы отправляете запрос через интерфейс, он сохраняется в файл `last-request.json`
2. AI (в Cursor) может прочитать этот файл и обработать запрос
3. Все запросы также сохраняются в папке `ai-requests/` для истории

### Обработка запросов AI

После отправки запроса через интерфейс, вы можете сказать AI в Cursor:
- "Обработай последний запрос из интерфейса"
- "Прочитай last-request.json и выполни запрос"
- "Выполни запрос из файла last-request.json"

AI прочитает файл и выполнит запрос, изменив проект соответственно.
