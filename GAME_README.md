# 🎨 Игра "Рисуй и Угадывай" - Многопользовательская Игра

## 🎮 Описание

Интересная многопользовательская игра, где один игрок рисует слово, а другие игроки пытаются его угадать!

## ✨ Возможности

- 🎨 **Интерактивное рисование** - рисуйте на canvas с настраиваемыми цветами и размером кисти
- 🎯 **Угадывание слов** - попробуйте угадать, что рисует другой игрок
- ⏱️ **Таймер** - ограниченное время на рисование и угадывание
- 🏆 **Система очков** - зарабатывайте очки за правильные ответы
- 📊 **Таблица лидеров** - следите за своими достижениями
- 🔄 **Множественные раунды** - играйте сколько угодно раундов

## 🚀 Как играть

1. Откройте `game.html` в браузере или перейдите по ссылке с главной страницы
2. Введите ваше имя
3. Нажмите "Подключиться"
4. Дождитесь начала игры
5. Когда наступит ваша очередь рисовать:
   - Вы увидите слово, которое нужно нарисовать
   - Используйте мышь для рисования на canvas
   - Настройте цвет и размер кисти
   - Постарайтесь нарисовать так, чтобы другие угадали!
6. Когда другой игрок рисует:
   - Смотрите на рисунок в реальном времени
   - Вводите ваши догадки
   - Первый угадавший получает очки!

## 🎯 Правила

- У вас есть 60 секунд на рисование
- Угадывающие имеют 30 секунд на угадывание
- За правильный ответ: +100 очков
- Игра продолжается раундами

## 🛠️ Технические детали

- **HTML5 Canvas** для рисования
- **JavaScript** для игровой логики
- **Responsive дизайн** - работает на всех устройствах
- Готово к интеграции с WebSocket для настоящего многопользовательского режима
