# 🔍 Как работает определение профессии

## Алгоритм анализа

Приложение использует анализ изображения с камеры для определения подходящей профессии. Вот как это работает:

### 1. Захват изображения
- Камера делает снимок текущего кадра
- Изображение сохраняется в canvas для анализа

### 2. Анализ характеристик изображения

Система анализирует три основных параметра:

#### 📊 Яркость (Brightness)
- Вычисляется средняя яркость всех пикселей
- Формула: `(R + G + B) / 3` для каждого пикселя
- Нормальное значение: 120-180 (хорошее освещение)

#### 🎨 Цветовое разнообразие (Color Variance)
- Измеряет разницу между цветовыми каналами (R, G, B)
- Формула: `|R-G| + |G-B| + |B-R|` для каждого пикселя
- Высокое значение (>30) = разнообразие цветов

#### ⚡ Контраст (Contrast)
- Разница между самым темным и самым светлым пикселем
- Формула: `maxBrightness - minBrightness`
- Высокое значение (>50) = хороший контраст

### 3. Система оценки профессий

Для каждой профессии вычисляется оценка на основе:

#### Базовые критерии:
- **Нормальное освещение** (120-180): +20 баллов
- **Разнообразие цветов** (>30): +15 баллов
- **Хороший контраст** (>50): +15 баллов
- **Случайный фактор**: +0-30 баллов (для разнообразия)

#### Специфические критерии для профессий:

- **Программист**: яркость 100-255, контраст >40 → +20
- **Дизайнер**: цветовое разнообразие >40 → +25
- **Врач**: яркость 110-160 → +20
- **Учитель**: яркость >100, контраст >30 → +20
- **Предприниматель**: яркость >120 → +20

### 4. Выбор профессии

- Все профессии получают оценку от 0 до 100+
- Профессии сортируются по убыванию оценки
- Выбирается профессия с наивысшей оценкой
- Процент совпадения: 65-95% (ограничено для реалистичности)

### 5. Альтернативные профессии

- Показываются топ-3 профессии после основной
- Их оценки: 50-90%

## ⚠️ Важно понимать

Это **упрощенный алгоритм** для демонстрации. В реальности определение профессии требует:
- Психологического тестирования
- Анализа навыков и интересов
- Оценки личностных качеств
- Профессионального консультирования

Текущий алгоритм основан на визуальных характеристиках изображения и носит **развлекательный/демонстрационный характер**.
